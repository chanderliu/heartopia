Heartopia Gift Codes & Redemption Guide ​

Working, verified Heartopia gift codes updated daily — redeem for Wishing Stars, Moonlight Crystals, and seasonal rewards.

Updated — View All Codes

How to redeem: Settings → Gift Code → Enter code → Confirm

How to Redeem Codes ​

How to redeem:

1
Open the Game

Launch Heartopia and log into your account.

2
Open Settings

Tap the gear icon in the top-right corner of the main screen.

3
Find Gift Code

Select "Gift Code" from the settings menu.

4
Enter the Code

Type or paste the code exactly as shown — case-sensitive.

5
Confirm & Claim

Tap Confirm — rewards are sent directly to your inventory.

Why Do Codes Stop Working? ​

  • Time limits — Most codes have expiration dates. Some last days, others last months.
  • Typing errors — Codes are case-sensitive. Copy-paste is safer than typing.
  • Regional restrictions — Some codes are region-locked to specific servers.
  • Already redeemed — Each code can only be redeemed once per account.
